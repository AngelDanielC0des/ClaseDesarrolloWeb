HACED UNA PÁGINA PARA QUE EL USUARIO JUEGUE A ADIVINAR UN NÚMERO

AL INICIO DEL PROGRAMA, EL ORDENADOR, PIENSA UN NÚMERO ALEATORIO ENTRE 1 Y 100, 
QUE EL USUARIO DEBERÁ ADIVINAR EN UN MÁXIMO DE 5 INTENTOS

POR CADA INTENTO, SI EL USUARIO ACIERTA, SE LE MUESTRA UNA IMAGEN DE FELICITACIÓN
SI SE AGOTAN SUS VIDAS Y NO HA ACERTADO, SE LE MUESTRA UNA IMAGEN DE DERROTA

SI FALLA, PERO LE QUEDAN INTENTOS, EL PROGRAMA DEBE INFORMAR AL USUARIO
DE SI EL NÚMERO QUE INTRODUJO  ES MAYOR O MENOR QUE EL NÚMERO BUSCADO
