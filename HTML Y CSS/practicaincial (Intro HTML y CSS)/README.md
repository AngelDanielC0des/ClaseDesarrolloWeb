# Lección: Introducción Inicial a HTML5 y CSS

Esta es la primera lección práctica del curso, diseñada para asentar las bases estructurales de la web (HTML) y entender cómo modificar la presentación visual básica (CSS).

## 🎯 Objetivos de la Lección
1. Comprender la estructura jerárquica elemental de un archivo HTML (`<html>`, `<head>`, `<body>`).
2. Entender y utilizar etiquetas semánticas básicas (`<header>`, `<main>`, `<nav>`, `<footer>`).
3. Crear listas ordenadas/desordenadas (`<ul>`, `<li>`) y tablas de datos estructuradas (`<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>`).
4. Trabajar con enlaces internos (anclas `#id`) y externos (URLs absolutas).
5. Comprender las alturas relativas del navegador (`vh`) y centrar elementos mediante el modelo Flexbox inicial.
6. Diferenciar entre el comportamiento de visualización en bloque e inline-block.

## 📂 Archivos del Ejercicio
* **`index.html`:** Contiene el esqueleto HTML5 del sitio, la barra de navegación, las tarjetas de lecciones y una sección de enlaces, todo ricamente documentado.
* **`styles/styles.css`:** Define los estilos del viewport, el menú en flexbox horizontal y los bloques alineados en inline-block con sombras y tablas estilizadas.
