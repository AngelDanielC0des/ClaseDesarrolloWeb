# Lección: Tablero Tres en Raya — Flexbox CSS

Esta lección enseña a usar el modelo de Caja Flexible (Flexbox) para distribuir y alinear elementos en una y dos dimensiones de forma fluida.

## 🎯 Objetivos de la Lección
1. Comprender los conceptos de contenedor flexible (`display: flex`) y elementos flexibles.
2. Manejar la dirección del flujo de caja mediante `flex-direction` (filas y columnas).
3. Entender los conceptos de Eje Principal (Main Axis) y Eje Cruzado (Cross Axis).
4. Distribuir el espacio en el eje principal usando `justify-content` (`center` y `space-between`).
5. Alinear elementos en el eje cruzado usando `align-items: center`.
6. Crear figuras geométricas redondas a partir de elementos cuadrados mediante `border-radius: 50%`.

## 📂 Archivos del Ejercicio
* **`index.html`:** Estructura del tablero con filas y fichas circulares representadas mediante divs semánticos comentados.
* **`styles.css`:** Contiene las reglas Flexbox que dan forma al tablero, diagramas de ejes en los comentarios para facilitar la comprensión.
