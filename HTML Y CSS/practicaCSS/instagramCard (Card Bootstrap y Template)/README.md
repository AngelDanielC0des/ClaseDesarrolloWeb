# Lección: Card de Instagram (Bootstrap, CSS Custom y Template HTML)

Esta lección combina el uso de un framework CSS externo (Bootstrap) con estilos propios para reproducir un componente tipo "card" de red social, e introduce la potente etiqueta `<template>` de HTML5.

## 🎯 Objetivos de la Lección
1. Integrar Bootstrap y Bootstrap Icons desde un CDN externo.
2. Sobrescribir y personalizar estilos de Bootstrap con CSS propio.
3. Construir un componente de card visual con Flexbox (cabecera, imagen, acciones, contenido).
4. Aplicar el truco del **margin negativo** para solapar avatares de usuarios.
5. Entender la etiqueta `<template>` como contenedor de HTML inerte (invisible), y clonar su contenido dinámicamente con JavaScript.
6. Usar atributos de accesibilidad (`aria-label`) en botones con solo iconos.

## 📂 Archivos del Ejercicio
* **`index.html`:** Estructura semántica del post de Instagram con `<article>`, `<figure>`, `<template>`, `<time>` y JavaScript para clonación dinámica de comentarios.
* **`style.css`:** Maquetación Flexbox de la card, avatares solapados, reset de botones nativos y estilos inspirados en Instagram.
