¿Cómo funciona la propiedad transition?

La transición CSS permite cambiar valores de propiedades de forma suave (durante un tiempo dado), en lugar de que el cambio sea inmediato.

¿Qué ventajas tiene position: sticky?

Combina relative y fixed. El elemento se comporta como relativo hasta que alcanza un umbral de scroll (ej. top: 0), momento en el cual se fija en la pantalla.
