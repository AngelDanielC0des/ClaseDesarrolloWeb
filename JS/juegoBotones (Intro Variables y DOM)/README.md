# Lección: Introducción Temprana a Variables y Manipulación del DOM

Esta lección marca el primer paso en la interacción entre el usuario (HTML) y la lógica de programación (JavaScript).

## 🎯 Objetivos de la Lección
1. **Entorno Global:** Familiarizarse con los objetos nativos del navegador (`window`, `document`, `console`).
2. **Selección de Elementos:** Usar el método fundamental `document.getElementById("id")` para capturar nodos del árbol HTML.
3. **Modificación de Estilos Dinámicos:** Cambiar propiedades visuales "al vuelo" a través de la propiedad `.style` (ej: `style.backgroundColor`, `style.display`).
4. **Propiedades del Navegador:** Leer la URL actual mediante `window.location.href` y las dimensiones del _viewport_ con `window.innerWidth` e `innerHeight`.
5. **Eventos Básicos:** Enlazar botones con código JavaScript mediante el atributo en línea `onclick="..."`.

## 📂 Archivos del Ejercicio
* **`index.html`:** Disposición en bloque de tres botones, cada uno vinculado a una función JS.
* **`index.js`:** Lógica de manipulación del DOM dividida en tres funciones simples (Saludar, Mostrar Info, Ocultar).
