# Lección: Temporizadores y Asincronía Básica (setTimeout)

Esta lección es una introducción fundamental a la programación asíncrona en el navegador, enseñando cómo retrasar la ejecución de bloques de código.

## 🎯 Objetivos de la Lección
1. **Comprender el Tiempo en JS:** Aprender a usar la función nativa `setTimeout()`, que pertenece a las Web APIs del navegador.
2. **Sintaxis de Callback:** Entender cómo pasar una función como parámetro a otra función (Callback). 
3. **Arrow Functions:** Utilizar la sintaxis moderna de "Función Flecha" `() => {}` para escribir código más compacto.
4. **Conversión de Tiempo:** Acostumbrarse a que en JavaScript el tiempo se mide en milisegundos (`5000ms = 5s`).
5. **Control Adicional (Anexo en JS):** Conocer la existencia de `clearTimeout()` para cancelar temporizadores antes de que lleguen a ejecutarse.

## 📂 Archivos del Ejercicio
* **`index.html`:** Página HTML básica que invoca al script externo.
* **`script.js`:** Un _one-liner_ (código de una sola línea) que desencadena una alerta del navegador después de una espera exacta de 5 segundos, complementado con notas teóricas adicionales.
