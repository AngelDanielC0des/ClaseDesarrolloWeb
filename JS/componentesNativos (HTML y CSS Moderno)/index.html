1. Popover API (Sin JavaScript)

Estos popovers funcionan SOLO con HTML, sin una línea de JS

Popover Automático

Este popover se cierra automáticamente al hacer clic fuera o presionar Escape.

Atributo: popover (sin valor = automático)

Popover Manual

Este popover SOLO se cierra con el botón de cerrar.

Atributo: popover="manual"

Popover Toggle

El botón alterna entre mostrar y ocultar.

Atributo: popovertargetaction="toggle"

Código HTML:

<button popovertarget="mi-popover">Abrir</button>
<div id="mi-popover" popover>
    <p>Contenido del popover</p>
</div>

2. CSS :has() - Parent Selector

Selecciona elementos PADRE basándose en sus HIJOS

Tarjeta sin imagen

Esta tarjeta no tiene imagen, así que tiene un estilo diferente.

Tarjeta con imagen

Esta tarjeta tiene imagen, así que el padding es mayor y el borde es azul.

Tarjeta con botón deshabilitado

El botón está deshabilitado, así que la tarjeta tiene opacidad reducida.

Tarjeta con enlace externo

Esta tarjeta tiene un enlace externo, así que tiene un icono especial.

Visitar sitio ↗

Código CSS con :has():

/* Tarjeta que CONTIENE una imagen */
.card:has(img) {
    border-color: #667eea;
    padding: 30px;
}

/* Tarjeta que CONTIENE un botón deshabilitado */
.card:has(button:disabled) {
    opacity: 0.6;
}

/* Formulario que CONTIENE un input inválido */
form:has(input:invalid) {
    border-color: red;
}

Código CSS:

/* Definir el contenedor */
.card-container {
    container-type: inline-size;
    container-name: card;
}

/* Estilos basados en el tamaño del CONTENEDOR */
@container card (min-width: 400px) {
    .product-card {
        flex-direction: row;  /* Horizontal si el contenedor es ancho */
    }
}

@container card (max-width: 399px) {
    .product-card {
        flex-direction: column;  /* Vertical si el contenedor es estrecho */
    }
}

4. CSS Nesting (Anidamiento Nativo)

Anida selectores CSS sin preprocesadores

  • Inicio
  • Productos
  • Contacto

Código CSS con Nesting:

/* ANTES (sin nesting) */
.nav { background: #333; }
.nav ul { list-style: none; }
.nav ul li { display: inline; }
.nav ul li a { color: white; }
.nav ul li a:hover { color: #667eea; }
.nav ul li a.active { font-weight: bold; }

/* AHORA (con nesting nativo) */
.nav {
    background: #333;
    
    ul {
        list-style: none;
        
        li {
            display: inline;
            
            a {
                color: white;
                
                &:hover {
                    color: #667eea;
                }
                
                &.active {
                    font-weight: bold;
                }
            }
        }
    }
}

📊 Resumen de Features Modernas

Popover API

HTML 2023

Crea popovers/modales sin JavaScript

  • popover - Popover automático
  • popover="manual" - Solo se cierra con botón
  • popovertarget - Botón que abre el popover

CSS :has()

CSS 2023

El "parent selector" más esperado

  • Selecciona padres según sus hijos
  • Funciona con cualquier selector
  • Reemplaza mucho JavaScript

Container Queries

CSS 2023

Responsive basado en contenedor

  • container-type: inline-size
  • @container (min-width: X)
  • Componentes verdaderamente reutilizables

CSS Nesting

CSS 2024

Anidamiento nativo sin preprocesadores

  • Usa & para referenciar al padre
  • Mejora la legibilidad
  • Ya no necesitas SASS/LESS
