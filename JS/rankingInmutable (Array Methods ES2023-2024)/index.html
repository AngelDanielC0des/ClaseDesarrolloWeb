Comparación: Mutable vs Inmutable

❌ Métodos mutables (antiguos)

const arr = [3, 1, 2];
arr.sort();
// arr ahora es [1, 2, 3]
// ¡El original se perdió!

✅ Métodos inmutables (ES2023)

const arr = [3, 1, 2];
const sorted = arr.toSorted();
// arr sigue siendo [3, 1, 2]
// sorted es [1, 2, 3]
