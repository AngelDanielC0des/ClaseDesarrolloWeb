Posts desde API

Prueba de HMR (Hot Module Replacement)

Modifica src/utils.js o src/style.css mientras el servidor está activo y observa cómo los cambios se reflejan al instante sin recargar la página.

Comandos del Proyecto

npm install

Instala todas las dependencias listadas en package.json dentro de node_modules/

npm run dev

Inicia el servidor de desarrollo de Vite con HMR (recarga instantánea)

npm run build

Genera la versión optimizada para producción en la carpeta dist/

npm run preview

Sirve localmente la versión de producción para verificar antes de desplegar
