# Lección: Adivina Adivinanza (Palíndromos JS)

Esta lección es un ejercicio de lectura de código legado, ingeniería inversa y pensamiento algorítmico.

## 🎯 Objetivos de la Lección
1. **Comprensión de Código Legado:** Leer un script sin nombre descriptivo y deducir su propósito funcional.
2. **Algoritmia Básica:** Entender el funcionamiento de un bucle `for` inverso que decrementa su contador (`i--`) para recorrer un _string_ de fin a principio.
3. **Refactorización Limpia:** Dar nombres semánticos a las funciones (renombrar el "misterioso" método a `esPalindromo`).
4. **Pensamiento Lateral (Anexo en `script.js`):** Descubrir que un problema en programación siempre tiene múltiples soluciones, proponiendo metodologías adicionales:
   * **Moderna/Declarativa:** Utilizando encadenamiento de métodos de arrays (`split.reverse.join`).
   * **Recursiva:** Invocando a la misma función con un fragmento recortado de la palabra base hasta alcanzar el caso base.

## 📂 Archivos del Ejercicio
* **`adivina.html`:** Interfaz web básica con la lógica del bucle inverso embebida y completamente comentada.
* **`script.js`:** Archivo anexo que incluye las soluciones a los "Retos Extra" planteados por el código original (versión de métodos de array y versión recursiva).
