# Lección: Diseño Landing Page (Tailwind CSS)

Esta lección enseña a estructurar una página de destino (Landing Page) moderna utilizando el framework CSS de utilidad Tailwind CSS cargado desde CDN.

## 🎯 Objetivos de la Lección
1. Aprender a integrar Tailwind CSS mediante CDN y configurar su tema de forma dinámica usando JavaScript (`tailwind.config`).
2. Diseñar un Grid asimétrico (Bento Grid) para maquetar secciones visuales.
3. Utilizar un Grid de 12 columnas (`grid-cols-12`) combinando anchos mediante `col-span-*`.
4. Utilizar filtros de fondo en elementos fijos (`backdrop-blur-md`).
5. Estructurar layouts responsivos aplicando modificadores de tamaño de pantalla (ej. `hidden md:flex`).
6. Manejar la opacidad del color de fondo con Tailwind (`bg-surface/80`).

## 📂 Archivos del Ejercicio
* **`index.html`:** Estructura completa de la landing mística "Mystic Void" con comentarios detallando las clases de Tailwind y sus funcionalidades responsivas.
* **`index2.html` / `index2(figma).html`:** Variaciones y experimentos de maquetación para la lección de landing page.
