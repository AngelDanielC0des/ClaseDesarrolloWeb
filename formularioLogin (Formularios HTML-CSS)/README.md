# Lección: Formulario de Login Básico (HTML y CSS)

Esta lección cubre las bases de creación de formularios en HTML5 y su estilización estructural básica con CSS.

## 🎯 Objetivos de la Lección
1. Utilizar inputs básicos de texto (`type="text"`) y contraseñas ocultas (`type="password"`).
2. Comprender la utilidad de agrupar campos relacionados con la etiqueta `<fieldset>` e identificarlos con `<legend>`.
3. Entender la vinculación entre un texto descriptivo (`<label>`) y su campo de entrada mediante la propiedad `for`.
4. Estudiar el posicionamiento absoluto en CSS (`position: absolute`).
5. Trabajar con pseudo-clases de estado sencillas como `:hover` en inputs.
6. Aplicar text-shadows y bordes redondeados (`border-radius: 50%`) en botones.

## 📂 Archivos del Ejercicio
* **`formularioLogin.html`:** Formulario HTML5 con campos de usuario, contraseña, fieldset, checkboxes y botones de enviar/limpiar.
* **`style.css`:** Hoja de estilos con posicionamiento absoluto, anchos fluidos basados en `vw` y `vh`, y sombras básicas comentadas didácticamente.
